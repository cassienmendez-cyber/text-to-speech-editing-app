# Drive Mode

> Eyes on the road. Hands on the wheel. Capture the thought before it
> disappears.

Drive Mode enables safe manuscript review during commutes. It is a deliberately
constrained, hands-free experience: the manuscript plays as audio, and the
author captures revision ideas by voice without ever touching the screen.

---

## Activation

Supported methods:

- Voice activation
- Manual activation
- Bluetooth vehicle detection
- Android Auto integration
- Apple CarPlay integration
- Optional speed-based activation

---

## Interface

Display only essential controls:

- Play
- Pause
- Add Note
- Skip Forward
- Skip Backward

Features:

- Oversized controls
- Simplified display
- Automatic dimming

---

## Wake Phrase Support

Examples:

- "Hey StoryScribe"
- "Story Note"
- "Editor"

Users may customize wake phrases.

---

## Voice Commands

### Playback

- Pause narration
- Resume narration
- Stop narration
- Repeat sentence
- Skip ahead
- Skip back
- Increase speed
- Decrease speed

### Navigation

- Next chapter
- Previous chapter
- Go to Chapter [number]

### Notes

- Add note
- Save note
- Cancel note
- Bookmark scene
- Flag pacing issue
- Flag continuity issue
- Flag character issue

---

## Note Workflow

When the user says **"Add note,"** StoryScribe should:

1. Pause narration.
2. Save manuscript location.
3. Record the user's note.
4. Convert speech into text.
5. Preserve original audio.
6. Save the note.

---

## Driving Confidence Settings

Users must explicitly choose their preferred mode.

| Mode | Behavior |
| --- | --- |
| **BEGINNER** | Confirmation required for every note. |
| **STANDARD** | Confirmation only when transcription confidence is low. |
| **EXPERT** | Notes save automatically. |

---

## Quick Capture System

**Quick Note** — Immediately records short ideas, stored in an Inbox.

> Example: "Quick note. Increase tension here."

**Emergency Note** — Captures unexpected breakthrough ideas without
interrupting playback, preserving inspiration with minimal friction.

---

## Safety Restrictions During Driving

Safety takes precedence over convenience. While driving, disable:

- Manual manuscript editing
- Extensive menu navigation
- AI rewriting
- Complex dashboard interactions

---

## Drive Session Summary

Upon ending Drive Mode, StoryScribe reports:

- Listening duration
- Chapters completed
- Notes created
- Bookmarks added
- Unresolved revision count

Optional AI summaries may identify recurring revision themes.
